 **Цифровая грамотность — готовность и способность личности применять цифровые технологии уверенно, эффективно, критично и безопасно во всех сферах жизнедеятельности.** 

 Цифровая грамотностьважнейший навык XXI века, основа безопасности в информационном обществе. Формированию цифровой грамотности должно уделяться особое внимание наравне с читательской, математической и естественнонаучной грамотностью.
